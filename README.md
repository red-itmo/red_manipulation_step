#Installation

1. run git clone
```
https://github.com/red-itmo/red_manipulation_step.git
```

2. Place "matrix" library to /usr/include (https://github.com/PX4/Matrix)

3. run ```catkin_make```

# How to run?

1. Start test
```
roslaunch arm_manipulation trajectory_test.launch
```
